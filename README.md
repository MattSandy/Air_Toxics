Air Toxics
==========

**To run from R:**  
Grab these pacakges
```R    
    require("dplyr")  
    require("devtools")  
    devtools::install_github("shiny", "rstudio")  
    devtools::install_github('rCharts', 'ramnathv')
```
Then use `runGitHub` from `devtools`  
```R
    library(shiny)`  
    runGitHub("Air_Toxics", "dKvale")
```
